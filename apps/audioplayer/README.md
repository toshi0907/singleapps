# audioplayer

音声ファイルをブラウザで再生できる Web アプリです。

## 前提環境

- Python 3.8 以上
- 外部依存なし（標準ライブラリのみ使用）

## 機能

- 音声ファイルのアップロード（MP3 / WAV / OGG / FLAC / AAC / M4A / WebM）
- あらかじめ `sounds/` に置いたファイルの再生
- 再生 / 一時停止 / 停止
- 音量変更（スライダー）
- 再生位置のシーク（プログレスバー）

## 使い方

### サーバーの起動

```sh
python src/server.py [ポート番号]
```

ポート番号を省略した場合は `8080` で起動します。

```sh
$ python src/server.py
サーバーを起動しました: http://localhost:8080
```

起動後、ブラウザで `http://localhost:8080` にアクセスします。

### プリロードファイルの配置

`sounds/` ディレクトリに音声ファイルを置くと、サーバー起動時から一覧に表示されます。

```sh
cp /path/to/music.mp3 sounds/
```

### ファイルのアップロード

ブラウザ画面下部の「ファイルをアップロード」フォームから音声ファイルを選択してアップロードできます。
アップロードされたファイルは `uploads/` ディレクトリに保存され、一覧に追加されます。

### 音声の再生

1. ファイル一覧からファイル名をクリックして選択
2. 再生が自動的に開始されます
3. ボタンで再生 / 一時停止 / 停止を操作できます
4. 音量スライダーで音量を調整できます
5. プログレスバーをドラッグして再生位置をシークできます

## テスト

```sh
python3 test/test_server.py
```

## 想定ユースケースと制約

- **ユースケース**: ローカルでシンプルに音声ファイルを試聴したい場合に便利です。
- **制約**:
  - 対応形式: `.mp3` `.wav` `.ogg` `.flac` `.aac` `.m4a` `.webm`
  - アップロードしたファイルは `uploads/` に保存されます（サーバー再起動後も残ります）
  - 複数ファイルの同時再生には対応していません
