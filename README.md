# singleapps

単機能アプリを集めるリポジトリです。

## apps

- [apps/helloworld](apps/helloworld/README.md): `helloworld` と出力する単機能 CLI
- [apps/url2title](apps/url2title/README.md): URL を入力するとそのページのタイトルを返す CLI・Web・API アプリ
- [apps/vscode-language-extension](apps/vscode-language-extension/README.md): VSCode 言語拡張機能テンプレート（シンタックスハイライト・言語設定）
- [apps/audioplayer](apps/audioplayer/README.md): 音声ファイルをブラウザで再生できる Web アプリ（再生・一時停止・停止・音量変更・シーク対応）
